Sinon.JS

sinon-nodeunit

nodeunit adapter for Sinon.JS. Automatic sandboxing and mock verification with no scaffolding.

The nodeunit adapter is discontinued. The adapter could not be created without modifying nodeunit, and I don't want to maintain a hacky adapter any longer. If you're interested in maintaining an adapter for nodeunit and Sinon, please get in touch. In the meantime, use Sinon.JS as a library with nodeunit and do your own sandboxing (it's not that hard).

If you are looking for a Node.JS test framework with good Sinon.JS support, check out newly released Buster.JS, developed by Sinon.JS author Christian Johansen and August Lilleaas.

Sinon uses Semantic versioning.
